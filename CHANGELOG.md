## [1.0.0] - 2026-02-27

### Added
- Core survival shooter gameplay
- Wave-based enemy spawning system
- Boss battles every 10 waves
- XP and leveling system with quadratic progression
- Upgrade rarity system (Common, Rare, Epic, Legendary)
- Pressure system for risk-reward mechanics
- Elite enemy spawning
- Risk-based XP bonuses
- Post-Wave 5 difficulty scaling
- Level fatigue mechanic
- Visual effects and particles
- Health and XP bars
- Score tracking
- Help panel with instructions
- Responsive UI design

### Enemy Types
- Basic enemies (standard)
- Fast enemies (quick movement)
- Tank enemies (high health)
- Elite enemies (tough variants)
- Boss enemies (multi-phase fights)

### Upgrade Categories
- Common: Small stat increases (+5% damage, speed, etc.)
- Rare: Noticeable improvements (piercing shots, +15% speed)
- Epic: Build-defining upgrades (explosive rounds, double shot)
- Legendary: Game-changing powers (chain lightning, critical strikes)

### Systems
- Pressure-based difficulty scaling
- Risk-reward XP multipliers
- Wave progression mechanics
- Visual feedback systems
- Particle effects
- Screen shake and hit flash

### Technical Features
- 60 FPS target framerate
- Efficient collision detection
- Optimized rendering pipeline
- Memory-efficient particle systems
- Clean code architecture
- Modular design for easy expansion

### UI/UX
- Clean, modern interface
- Visual upgrade cards with rarity indicators
- Real-time health and XP display
- Wave progression indicator
- Sound toggle controls
- Help system
- Game over screen with restart

### Balance
- Fair difficulty progression
- Strategic upgrade choices
- Risk vs. reward mechanics
- Multiple viable playstyles
- Skill expression through movement and positioning
